# FarMeet 機能ロードマップ

> 顧客体験を向上させるための優先度付きタスク一覧

## 📊 現在の実装状況

### ✅ 完了済み機能
- 認証（Email/Googleログイン）
- 農園一覧・詳細・検索・フィルター（場所・日程・人数・価格帯・カテゴリ）
- 予約フロー・管理
- お気に入り機能
- レビュー投稿・表示
- 決済（Stripe/PayPay）
- 管理者ダッシュボード
- 農家ダッシュボード
- 最近見た農園
- OGP対応（SNSシェア）

## 🚀 優先度別タスク一覧

### 🔴 P0: 高優先度（顧客体験に直結）

| # | 機能 | 説明 | 期待効果 |
|---|------|------|----------|
| 1 | **予約リマインダー通知** | 予約日前日にメール通知を送信 | キャンセル/No-show削減 |
| 2 | **予約確認メール** | 予約完了時に詳細付きメール送信 | 安心感向上、問い合わせ削減 |
| 3 | **農家とユーザー間メッセージ** | 予約前後の質問・連絡機能 | 顧客満足度向上 |
| 4 | **レビュー投稿促進** | 体験後にレビュー依頼メール送信 | レビュー数増加→信頼性向上 |

## 📌 次のアクション候補

> [!TIP]
> まずは **P0: 予約関連の通知機能** から始めることを推奨します。
> メール通知は顧客の安心感に直結し、No-show削減にも効果的です。

### すぐに着手できるタスク

1. **予約確認メール** - Spring Bootのメール機能実装
2. **予約リマインダー** - Spring Schedulerでバッチ処理
3. **レビュー依頼メール** - 体験後自動送信

## 🛠️ 技術的考慮事項

| 機能 | 必要な技術/サービス |
|------|-------------------|
| メール通知 | Spring Mail + SMTP (SendGrid/AWS SES等) |
| メッセージ機能 | WebSocket or ポーリング + 新規テーブル |
| 地図表示 | Google Maps JavaScript API |
| レコメンド | 協調フィルタリング or シンプルなルールベース |
